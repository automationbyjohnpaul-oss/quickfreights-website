# Purpose

This document defines the Search Engine Optimization (SEO) strategy for the Quick Freights Global Limited website.

The objective is to improve visibility for customers searching for customs clearing, freight forwarding, cargo handling, and logistics services within Nigeria, while establishing long-term organic search growth.

This document serves as the SEO reference for future content development and website enhancements.

---

# SEO Objectives

The website has four primary SEO goals:

- Improve visibility for customs clearing services
- Rank for freight forwarding and logistics searches
- Generate qualified business enquiries
- Build long-term domain authority

Success will be measured through increased organic traffic, qualified leads, and search rankings for targeted keywords.

# Metadata Standards

Every page should contain:

- Unique title
- Unique meta description
- Canonical URL
- Open Graph tags
- Twitter Card tags (future)
- Favicon
- Manifest

---

## Page Titles

Recommended structure:

```
Page Name | Quick Freights Global Limited
```

Examples:

```
Home | Quick Freights Global Limited

About | Quick Freights Global Limited

Services | Quick Freights Global Limited

Cargo Portal | Quick Freights Global Limited

Contact | Quick Freights Global Limited
```

---

## Meta Description

Recommended length:

150–160 characters.

Example:

> Professional customs clearing, freight forwarding, and cargo logistics services in Nigeria. Trusted support for imports, documentation, and port clearance.

---

# Heading Structure

Every page should have:

One H1 only.

Example:

```
H1

Moving Cargo.
Building Trust.
```

Subsections should use:

- H2
- H3
- H4

Never skip heading levels unnecessarily.

---

# Internal Linking Strategy

Every major page should link naturally to:

- Services
- Cargo Portal
- Contact
- About

Service descriptions should reference related services where appropriate.

Future blog articles should also link back to service pages.

---

# Image SEO

Every image should include:

Meaningful filename.

Example:

```
customs-clearance-onne-port.webp
```

Instead of:

```
IMG001.webp
```

---

Every image should also contain descriptive alt text.

Example:

```
Customs clearing staff assisting cargo documentation at Onne Port.
```
